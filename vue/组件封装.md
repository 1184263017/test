# 组件封装

## 创建项目

### vue create bs-ui
- Babel
- CSS Pre-processors

### 安装 vscode 插件

1. ESLint 在代码实时显示错误
2. Prettier ESLint 修正格式化代码

## 下载安装bootstrap
`npm install bootstrap`

导入样式
`import 'bootstrap/dist/css/bootstrap.min.css'`

测试
```html
<button class="btn btn-primary">hello</button>
```

## 封装bs-button

### 属性

- size  大小
- type 颜色
- outline 是否有边框
- disabled 是否禁用

### 插槽

- `#default` 或 `_` 默认插槽 设置按钮文本

### 事件

- click 单击事件
